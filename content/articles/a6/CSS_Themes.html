A Little About Themes


CSS Themes are pre-set visual styles and layout for a website. They apply to a website much like templates and themes apply to a slide deck in PowerPoint

Many CSS Frameworks have had themes developed for them. Themes allow for a change in the look and feel of a website without modification of the underlying structure of the CSS framework. This site makes use of two themes for the Bootstrap framework: Flatly (for the light theme), and Darkly (for the dark theme). You can see this theme change is action by switching the theme on this site.

Many sites and frameworks offer their own themes. Search for themes for the framework you're considering to see what is avaiable.
