Good Form!


HTML Forms are the main means of allowing users to send data to a web site.

HTML Forms are made up of form controls such as text fields, select boxes, buttons, checkboxes, and radio buttons.

Forms are specified in HTML with the <form></form> element. This element formally defines a form on a web page. It is a container element that supports some specific attributes to determine how a form behaves.

The code for a form with a name and an email input field looks like this:

        
<form method="post">
    <div>
        <label for="name">Name: </label>
        <input type="text" id="name" name="user_name">
    </div>
    <div>
        <label for="email">email: </label>
        <input type="email" id="email" name="user_email">
    </div>
</form>

Here, we can see the form code wrapped by the <form> element. The for attribute in the <label> tag contains the id of the <input> tag to which it is associated.

A good introduction to HTML forms is Mozilla's Your first HTML form tutorial.
