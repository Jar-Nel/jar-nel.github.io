HTML, not HoTMaiL


An HTML document is a tree of HTML nodes, or elements, that are declared using tags. An HTML element is an individual component of an HTML document.

An HTML element is labeled by placing content between an opening tag and a closing tag. The only difference betwen an opening tag and a closing tag is the forward slash "/" the closing tag contains.

The basic elements of an HTML page are:

  • Text headers, specified with the <h1></h1> <h2></h2> <h3></h3> <h4></h4> <h5></h5> <h6></h6> tags.
  • Paragraphs, specified with the <p></p> tag.
  • Divisions, specified with the <div></div> tag.
  • Links, specified with the <a></a> tags.
  • Images, specified with the <img>

Every element can have attributes. There are different sets of attributes for each element. There are also elements that are common to every element.

Some common attributes are:

  • id, wich gives an ID to an element in a page. Each ID must be unique on a page.
  • class, which is used to specify the CSS class style that is applied to an element.
  • style, which is used to give specific CSS styles to an element outside of a class.

When put together, an HTML tag with attributes will look something like this:
<span id="spanDemo" style="color: red">Some red text.</span>
When processed by a page it will look like this:
Some red text.

You can learn more about HTML elements and see how they are used by visiting Learn-HTML.org.
