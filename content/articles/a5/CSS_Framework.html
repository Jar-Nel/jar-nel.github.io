What is a CSS Framework?


A CSS Framework is a collection of CSS and sometimes JavaScript tha is meant to allow for easier web design, rapid development, and standards compliance.

Most frameworks contain a grid system to help with responsive layout. They also include tools for typography, imagry and iconography, forms elements, and graphical interface elements like tabs and slideshows.

This site makes use of a framework called Bootstrap. This framework uses a grid system for responsive design. It included the graphical elements that were used in the site's navigation bar and breadcrumb features at the top of the page. It also included layout elements like the card element in which this article appears.

This site also uses two themes for the Bootstrap framework that provide consistent coloring and typography across pages of the site.



Popular Frameworks

These are some popular frameworks used by web designers.

  • Bootstrap

    Used on this site and many others, Bootstrap is one of the most popular frameworks available. It's popularity means there is a lot of support available from an active user community.

    Bootstrap Website

  • Skeleton

    Skeleton is a lightweight, simple, responsive boilerplate framework that is used for smaller projects. The styles are designed to be used as a starting point. Skeleton is also a good fit beginning designers that are new to CSS frameworks.

    Skeleton Website

  • Pure.css

    Pure.css is designed to be a lightweight CSS framwrodk that stays out of the way. It supports responsive layouts and a collection of extensions to expand its functionality.

    Pure.css Website

  • Semantic UI

    Semantic UI makes use of natural language principles in an attempt to make code more readable.

    Semantic UI Website
