If you build it...


Here is a list of libraries and frameworks that were used in the construction and function of this website.

  • jQuery 3.3.1

    jQuery sells itself as a fast, small, feature-rich JavaScript library. Personally I have never found it to be particularly fast (or small), but Bootstrap requires it so here it is.

    jQuery Website

  • Bootstrap 4.3.1

    Bootstrap is an open source toolkit for developing with HTML, CSS, and JavaScript. It features a responsive grid system, prebuilt components, and jQuery plugins.

    Bootstrap Website

  • Bootstrap Themes: Flatly and Darkly

    Flatly and Darkly are Bootstrap themes from Bootswatch that are used for the light and dark modes of this site.

    Bootswatch Flatly   Bootswatch Darkly

  • Prism Syntax Highlighting

    A CSS and JavaScript library used for highlighting code used in code elements.

    Prism Website

  • Prism Themes: GHColors (light) and Atom Dark (Dark)

    Some syntax highlighting themes for use with Prism.

    Prism Themes

  • Marked.js

    A minimal JavaScript library for processing Markdown documents and converting them to HTML for viewing in a browser.

    Marked.js Website
