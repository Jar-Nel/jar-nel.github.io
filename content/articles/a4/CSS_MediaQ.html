Querying the Target

CSS Media Queries are a must for a truly responsive website.

A CSS media query will determine the output target of web content and allow the CSS to adjust accordingly so the content will look its best no matter where it is output.

CSS Media Queries can be used in responsive web design to target content to specific devices, from a larg screen display, to a computer, to a laptop, to a tablet, to a phone, even print. The CSS can be dynamically adjusted for each output type.

Media queries are defined using the @media rule followed by a media type. The available media types are all, print, screen, and speech. When no media type is specified all is implied.

Media features are defined in parentheses. Some of the most popular features are width, height, aspect-ratio, orientation, and resolution. Range features have a min and max version available.

In the following simple example, the body background color is black, but screen devices with a width that is 650px or less wil have a background color of blue instead:

        
body {
    background: black;
}

@media screen and (max-width: 650px) {
    body {
        background: blue;
    }
}

Mastering media queries can help you create a truly dynamic and responsive website without developing a dependence on heavy CSS frameworks and their limited grid systems.
