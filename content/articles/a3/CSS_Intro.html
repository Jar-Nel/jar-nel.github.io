Stylin' Sheets


CSS stands for Cascading Style Sheets. It is used to add style to web page elements. CSS can be used to alter the size, color, font, spacing, position, and many other visual items of your content.

A set of CSS rules contained within a document called a stylesheet is used to determine how a web page should look.

It can be difficult to visualize, so lets try an example that was found on Mozilla's dev site:

        
.example {
    color: blue;
    background-color: yellow;
    border: 1px solid black;
}

Here we are specifying a class we are calling example. It contains three properties and their values. Each property and value pair is called a declaration.

The declarations in the example class do the following:

  1. The first declatation sets the text color to blue.
  2. The second sets the background color to yellow.
  3. The third puts a black border around the element that is one pixel wide and solid.

A CSS class is assigned to an HTML tag with the class attribute, like this:

<span class="example">This is blue text on a yellow background.</span>

When rendered by the browser the code will end up looking like this:

This is blue text on a yellow background.

For more information on CSS, how to use it, and what you can do with it, see How CSS works on Mozilla's developer network.
