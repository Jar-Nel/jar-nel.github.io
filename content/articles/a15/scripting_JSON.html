Notable Notation


JSON stands for JavaScript Object Notation. It is an open file format that uses human-readable text to transmit data. The data consists of attribute-value pairs and serialized data types. It is a common format used for client-server communication.

Although JSON was derived from Javascript it is a language-independent format. Many programming languaages now include code for generate and read JSON formatted data.

The media type for JSON is application/json and it used the file extension .json
